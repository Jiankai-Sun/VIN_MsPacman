# VIN_MsPacman
This project aims to find an approach to train VIN on MsPacman. VIN is similar to Deep Q learning. However, the current way the code implement probably is not correct. File `log.txt` records the results.

## Requirement
- OpenAI Gym 0.9.6
- Python 3.5
- PyTorch 0.3.1

## How to run
```
python train.py

python test.py
```
